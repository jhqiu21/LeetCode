## Solution 1 HashMap
### Approach



### Complexity
- Time complexity: $O(n)$
- Space complexity: $O(k)$ for k distinct element in array

### Code
``` java linenums="1"
class Solution {
    public int majorityElement(int[] nums) {
        HashMap<Integer, Integer> map = new HashMap<>();
        int n = nums.length;
        for (int i = 0; i < nums.length; i++) {
            int before = map.getOrDefault(nums[i], 0);
            if (before == n / 2) {
                return nums[i];
            }
            map.put(nums[i], before + 1);
        }
        return -1;
    }
}
```

``` java linenums="1"
class Solution {
    public int majorityElement(int[] nums) {
        int count = 0;
        int res = 0;
        for (int num : nums) {
            if (count == 0) {
                res = num;
            }
            if (num == res) {
                count++;
            } else {
                count--;
            }
        }
        return res;
    }
}
```
